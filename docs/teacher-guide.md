# 감성 톡톡 동시 스튜디오 수업 가이드(교사용)

이 가이드는 3~4학년 수업에서 바로 사용할 수 있는 40분 수업 운영안입니다.
실행 기준 앱: `감성 톡톡 동시 스튜디오: 내 목소리 오디오북`

## 1. 성취기준 연결 (3~4학년 국어/음악)

- 국어 `[4국05-01]` 시나 이야기를 읽고 생각이나 느낌을 다양한 방식으로 표현한다.
- 음악 `[4음01-01]` 악곡의 특징을 이해하며 노래 부르거나 악기로 연주한다.
- 수업 연결 예시: 시의 생각과 느낌을 낭송으로 표현하고, 소리의 특징을 비교해 작품 분위기에 맞는 사운드를 고른다.
  (3학년 수업에서는 난이도 조절을 통해 같은 흐름으로 운영하세요.)
- 수업은 `동시 제목` 입력, `낭송 녹음 시작`, `사운드 팔레트`, `사운드 미리듣기`, `오디오 엽서 저장`의 실제 UI 흐름을 그대로 따라갑니다.

## 2. 40분 수업 운영 플로우

### 도입 (7분)
- 수업 목표 안내: “동시의 분위기에 어울리는 소리를 골라 오디오 엽서를 만든다.”
- 앱 화면에서 **동시 제목**이 어떤 역할인지 보여주기
- 마이크 사용 전, 오늘의 조용한 규칙 합의(손들기, 발언권, 소리 레벨)
- 음성 미리듣기 데모 1회: `낭송 녹음 시작` → `녹음 정지` → `내 목소리 들어보기`

### 활동 (23분)
- 1인 또는 짝 활동으로 진행(각 학생/조별)
- 학생은 아래 순서로 진행
  1. `동시 제목` 입력
  2. 동시를 읽고 `낭송 녹음 시작` 눌러 낭송
  3. `녹음 정지`, `내 목소리 들어보기`
  4. `다시 녹음`으로 품질/호흡/속도 조절
  5. `사운드 팔레트`에서 1~3개 선택
     - `빗소리`, `잔잔한 피아노`, `파도 소리`, `새소리`
  6. `사운드 미리듣기`로 최종 분위기 조율
  7. `오디오 엽서 저장` 클릭해 `.wav` 저장
- 교사 이동 포인트: 잘 안 되는 학생은 `사운드 팔레트` 개수 줄이기, 한 줄씩 천천히 읽기 지도

### 공유/발표 (7분)
- 저장된 `.wav`를 학생 또는 조별로 1곡씩 즉시 재생
- 발표 피드백 문구는 2개 기준으로 고정:
  1) 낭송의 감정선이 들리는가?
  2) 배경음이 시의 분위기를 더 잘 보여주는가?
- 발표 전후로 교사가 “오늘 잘한 점 1개 + 보완점 1개”만 언어화해 과잉 비판 방지

### 정리 (3분)
- 전 클래스 투표: “가장 적절한 소리는 어떤 조합이었나?”
- 과제 힌트 제공: 다음 시간엔 제목은 유지하고 소리만 바꿔 재도전
- 학생 음성 파일 보관 정책 안내 후 수업 종료

## 3. 학생용 활동지(인쇄/디지털)

### A안: 활동지 문항
1. 오늘의 시 제목은 무엇인가요?
2. 시의 핵심 느낌(차분함/행복함/그리움/긴장감) 한 가지를 고르세요.
3. 내 낭송에서 가장 잘 전달된 단어는 무엇인가요?
4. 어떤 소리를 골랐나요? (1개 이상)
5. 선택한 소리가 시의 분위기와 맞는 이유를 한 문장으로 쓰세요.
6. `내 목소리 들어보기` 후 바꾼 점을 한 가지 적어보세요.

### 관찰 체크(수업 중 메모용)
- 문장 흐름이 끊기지 않게 낭송했는가?
- 소리를 1~3개만 깔끔하게 사용했는가?
- 미리듣기 후 반복 개선(다시 녹음/사운드 조정) 시도했는가?
- 발표 후 피드백을 반영해 한 가지는 수정했는가?

## 4. 교사용 관찰/평가 체크포인트

- [ ] 동시 제목을 정확히 입력했는가
- [ ] 1회 낭송 후 부족한 부분을 파악하고 `다시 녹음`을 시도했는가
- [ ] 낭송 속도가 시의 정서와 맞는가
- [ ] 사운드 선택이 텍스트 정서를 살리는가
- [ ] `사운드 미리듣기`를 사용해 저장 전 점검했는가
- [ ] 발표 발언 시 감정표현과 자기평가가 일치하는가

### 배점 예시(총 10점)
- 낭송 몰입과 표현력 4점
- 사운드 매칭 적절성 3점
- 반복 수정 태도 2점
- 발표 정리 설명 1점

## 5. 저작권·개인정보·소음 관리

- 시 텍스트 사용:
  - 교과서/워크시트의 제공 텍스트는 교육 목적 범위에서 사용하고, 외부 텍스트를 추가할 경우 출처와 저작권을 확인하세요.
  - 직접 편집한 오디오(수록한 음악/효과음)는 라이선스가 명시된 파일만 사용하세요.
- 개인정보:
  - 저장은 **브라우저 내 처리 후 즉시 다운로드**되므로 앱 서버로 자동 업로드되지 않습니다.
  - 단, 기기 다운로드 폴더가 iCloud/Drive/OneDrive/학교 MDM 등과 동기화될 수 있으므로 수업 전 저장 위치와 학교 개인정보 지침을 확인하세요.
  - 학생 파일명은 민감 정보를 넣지 말고 수업일/반 위주로 정렬하세요(예: `3-2_동시활동_2조.wav`).
  - 수업 종료 후 학생별 저장본은 수업용 폴더에만 보관하고 불필요 파일은 즉시 삭제하세요.
- 소음 관리:
  - 낭송 구간은 “10분 간격 휴지 조용 신호(손들기/카운트다운)”를 고정해 반의 소음 패턴을 안정화하세요.
  - 1조씩 돌아가며 `낭송 녹음 시작` 데모를 진행하고, 다른 조는 모니터링 모드로 지켜보게 하세요.
  - 공유기기 수업에서는 가능하면 헤드셋·외부 스피커를 분리 사용해 재녹음 시 피드백 오염을 줄이세요.
- 기록/공유:
  - 학생 동의만으로 공유 범위를 판단하지 말고, 학교 개인정보 지침과 보호자/학교 승인 기준을 함께 확인하세요.
  - 공유가 승인된 파일만 **학교 승인된 비공개 클래스 채널**이나 **교사가 관리하는 수업 전용 폴더**에 업로드하세요. 공개 채널에는 올리지 마세요.

## 6. 비상 대응 체크

- 녹음이 되지 않을 때: 브라우저 권한 → `낭송 녹음 시작` 버튼 상태 → `마이크를 사용할 수 없습니다. 브라우저 권한을 확인해 주세요.` 메시지 순으로 확인
- 미리듣기가 안 될 때:  `내 목소리 들어보기`와 `사운드 미리듣기`를 교대로 테스트, 브라우저 캐시 새로고침
- 저장이 안 될 때: 브라우저 버전 및 다운로드 폴더 접근 권한 확인, 공용 기기에서는 저장 경로 공지
